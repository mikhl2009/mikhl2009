#  About Me:
I'm a software developer focused on building modern web applications using .NET and Azure. My work revolves around infrastructure automation, DevOps, and creating reliable CI/CD pipelines that make software delivery smoother and faster.<br><br>I'm always open to collaborating on projects that push the boundaries of cloud-based development, particularly in Azure and DevOps ecosystems. I appreciate working with others who value scalability, security, and thoughtful engineering.<br><br>Lately, I’ve been exploring ways to improve performance in large-scale .NET systems and refine DevOps workflows to be both efficient and maintainable. I enjoy learning new tools and approaches—right now, I’m deepening my understanding of Infrastructure as Code through Bicep and Terraform, as well as advanced automation within Azure DevOps.<br><br>Feel free to reach out if you want to talk about .NET development, DevOps practices, cloud architecture, or integrating CI/CD into your projects.<br><br>Outside of development, I have a genuine curiosity about technology as a whole—especially how it evolves. I'm also a big fan of science fiction and often find inspiration in the way it imagines the future of tech.
